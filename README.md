# 项目名：基于PyTroch+ResNet18的是否佩戴口罩识别



## 使用说明：

将待预测图片放入`./myDogCat_Classify_GoogLeNet/image_for_test/test_grape.jpg`。之后运行`./myDogCat_Classify_GoogLeNet/image_for_test/one_picture_predict.py`，就能得到运行结果。



## 模型与模型参数说明：

`./GoogLeNet_model.py`是构建好的GoogLeNet模型。

`./best_model.pth`是根据GoogLeNet模型训练好得到的模型参数。
